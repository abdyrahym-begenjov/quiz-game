# Quiz Game 🧠

## English
A simple command-line quiz game written in Python. It supports two different game modes, dynamically loads questions from a JSON file, and tracks your score.

### 🚀 Features
* **Two Game Modes:**
  * **Simple Mode:** Choose a specific number of questions to answer.
  * **Infinity Mode:** Play until you run out of lives (you start with 3 ❤️).
* **Randomized Questions:** Questions are picked randomly, and you won't get the same question twice in a single session.
* **Formatted UI:** Clean double-column layout for multiple-choice options (A, B, C, D).

### 🛠️ Requirements & Installation
1. Make sure you have Python installed.
2. The project requires a custom library propython (used for reading the JSON file via pyread) and a q1.json file containing the questions.
3. Run the game using:
   python Quiz.py


## Русский
Простая игра-викторина для командной строки, написанная на Python. Она поддерживает два различных игровых режима, динамически загружает вопросы из JSON-файла и ведет учет вашего счета.

### 🚀 Особенности
* **Два игровых режима:**
  * **Простой режим:** Выберите определенное количество вопросов, на которые хотите ответить.
  * **Бесконечный режим:** Играйте до тех пор, пока у вас не закончатся «жизни» (вы начинаете с 3 ❤️).
* **Случайный порядок вопросов:** Вопросы выбираются случайным образом, и в рамках одной игровой сессии вам не попадется один и тот же вопрос дважды.
* **Форматированный интерфейс:** Аккуратное двухколоночное оформление вариантов ответов (A, B, C, D).

### 🛠️ Требования и установка
1. Убедитесь, что у вас установлен Python.
2. Для работы проекта требуются специальная библиотека propython (используется для чтения JSON-файла посредством модуля pyread) и файл q1.json, содержащий вопросы.
3. Запустите игру с помощью команды:
   python Quiz.py
